Detailed Description

Core utility macros used across the BLOP project.

This header defines lightweight helpers for error reporting, assertions, abort/exit handling, pointer validation, memory allocation, and miscellaneous arithmetic utilities. All modules in BLOP rely on these common primitives.

Configuration Macros

  • Define BLOP_I_DONT_WANT_TO_CRASH_ON_ERROR_I_KNOW_WHAT_I_AM_DOING to disable abort/exit on fatal errors.
  • Define BLOP_I_DONT_WANT_TO_CRASH_ON_ERROR_I_KNOW_WHAT_I_AM_DOING to completely disable all BLOP assertions.
  • Define BLOP_SAFE_MODE to enable NULL pointer checks.
  • Define BLOP_ABORT_ON_ERROR to use abort() instead of exit(-1) on assertion.

These flags allow each user to fine-tune BLOP's safety vs performance tradeoffs.
