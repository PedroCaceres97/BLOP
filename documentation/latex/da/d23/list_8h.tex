\doxysubsection{Detailed Description}
Generic intrusive doubly-\/linked list (header-\/only, macro-\/configurable). 

This header implements a small, configurable, intrusive doubly-\/linked list. The module is designed to be lightweight, header-\/only, and fully customizable through macro configuration.\hypertarget{list_8h_autotoc_md1}{}\doxysubsubsection{\texorpdfstring{Overview}{Overview}}\label{list_8h_autotoc_md1}
The list is intrusive\+: each node holds its own data and the pointers linking it to the list. This reduces allocations and allows users to fully control memory layouts.

The API is generated from configuration macros and may produce multiple list types simply by including this header with different macro definitions.\hypertarget{list_8h_autotoc_md2}{}\doxysubsubsection{\texorpdfstring{How to use}{How to use}}\label{list_8h_autotoc_md2}
