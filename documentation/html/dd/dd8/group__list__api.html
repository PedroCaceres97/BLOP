◆ blopfn_list_destroy()

void blopfn_list_destroy ( blopt_list list)

Destroy a list structure.

The list must be empty (size == 0) when destroyed. Use blopfn_list_clear to remove and optionally deallocate nodes prior to calling destroy.

Parameters
listList to destroy.
