\doxysubsection{Detailed Description}
Configurable, header-\/only dynamic array (vector) implementation. 

This header emits a typed, configurable dynamic array API when included. Configure the generated type and behavior using the following macros prior to inclusion\+:


\begin{DoxyItemize}
\item BLOP\+\_\+\+VECTOR\+\_\+\+NAME\+: Identifier for the generated vector type and API (default\+: {\ttfamily vectorb}).
\item BLOP\+\_\+\+VECTOR\+\_\+\+DATA\+\_\+\+TYPE\+: Element type stored in the vector (default\+: {\ttfamily uint8\+\_\+t}).
\item BLOP\+\_\+\+VECTOR\+\_\+\+SCALATOR\+: Growth factor used when expanding capacity (default\+: 2; minimum is 2).
\item BLOP\+\_\+\+VECTOR\+\_\+\+INITIAL\+\_\+\+COUNT\+: Initial capacity when a vector is created (default\+: 10).
\item BLOP\+\_\+\+VECTOR\+\_\+\+DEALLOCATE\+\_\+\+DATA\+: Optional macro/function called to deallocate an element when removed (should accept one argument).
\item BLOP\+\_\+\+VECTOR\+\_\+\+IMPLEMENTATION\+: Define in exactly one translation unit to emit the function implementations. Without this only the declarations are produced.
\end{DoxyItemize}

The API is emitted under mangled names derived from BLOP\+\_\+\+VECTOR\+\_\+\+NAME. Use the produced symbols (functions/typedefs/macros) to manipulate the vector.
