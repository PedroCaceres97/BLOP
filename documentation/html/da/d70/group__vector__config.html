◆ BLOP_VECTOR_DATA_TYPE

#define BLOP_VECTOR_DATA_TYPE   uint8_t

Type stored in each element of the vector.

Default is uint8_t. To store ints, structs, or pointers set this before including the header.

◆ BLOP_VECTOR_NAME

#define BLOP_VECTOR_NAME   vectorb

Name used to build the public API identifiers.

Example: defining BLOP_VECTOR_NAME myvec will produce functions such as myvec_create, myvec_push_back, etc.

◆ BLOP_VECTOR_SCALATOR

#define BLOP_VECTOR_SCALATOR   2

Growth factor to use when increasing capacity.

Must be >= 2; if not defined or less than 2 the default 2 is used.
