\doxysubsection{Detailed Description}
Core utility macros used across the BLOP project. 

This header defines lightweight helpers for error reporting, assertions, abort/exit handling, pointer validation, memory allocation, and miscellaneous arithmetic utilities. All modules in BLOP rely on these common primitives.
