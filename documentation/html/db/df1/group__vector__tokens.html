◆ blopfn_vector_create

blopt_vector blopfn_vector_create ( )    BLOP_CONCAT2(blopt_vector, _create)

Allocate and initialize a new vector instance.

Function name tokens generated from the chosen vector name.

Behavior:

  • Allocates the vector structure and an internal buffer with BLOP_VECTOR_INITIAL_COUNT capacity.
  • On allocation failure prints an error and calls BLOP_ABORT (may exit/abort).
Returns
Pointer to the new vector, or NULL on failure (after BLOP_ABORT if configured).

#define blopt_vector   BLOP_VECTOR_NAME

Opaque pointer typedef name for the generated vector.

Opaque pointer to the generated vector type.

When BLOP_VECTOR_IMPLEMENTATION is defined the concrete struct is declared and this typedef resolves to struct blops_vector *.
