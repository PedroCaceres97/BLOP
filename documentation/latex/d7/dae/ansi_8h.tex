\doxysubsection{Detailed Description}
ANSI escape sequence helper macros. 

A lightweight, header-\/only collection of macros generating ANSI/\+VT100 escape sequences for terminal text styling, coloring, cursor control and screen manipulation. All macros expand into constant string literals that may be printed directly to any ANSI-\/compatible terminal.

Usage examples\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{printf(\mbox{\hyperlink{group__ansi__colors_ga4b82ce02685b3d7ebc5e8da4eaefae53}{ANSI\_FG\_RED}}\ \textcolor{stringliteral}{"{}Error!"{}}\ \mbox{\hyperlink{group__ansi__styles_gaf77a445894f2f750d43cf2182cd29e55}{ANSI\_RESET}});}
\DoxyCodeLine{printf(\mbox{\hyperlink{group__ansi__extended_ga83224b1892feed6a11a38ab803351222}{ANSI\_FG\_256}}(42)\ \textcolor{stringliteral}{"{}Hello"{}}\ \mbox{\hyperlink{group__ansi__styles_gaf77a445894f2f750d43cf2182cd29e55}{ANSI\_RESET}});}
\DoxyCodeLine{printf(\mbox{\hyperlink{group__ansi__cursor_gae9745d57d25f85d93b779d06631aff26}{ANSI\_CURSOR\_POS}}(10,\ 5));}

\end{DoxyCode}


Important\+:
\begin{DoxyItemize}
\item Many macros use the \# operator and require compile-\/time literal numbers.
\item These macros produce raw escape sequences (no validation is performed).
