\doxysubsection{Detailed Description}
Core utility macros used across the BLOP project. 

This header defines lightweight helpers for error reporting, assertions, abort/exit handling, pointer validation, memory allocation, and miscellaneous arithmetic utilities. All modules in BLOP rely on these common primitives.\hypertarget{blop_8h_autotoc_md0}{}\doxysubsubsection{\texorpdfstring{Configuration Macros}{Configuration Macros}}\label{blop_8h_autotoc_md0}

\begin{DoxyItemize}
\item Define {\ttfamily BLOP\+\_\+\+I\+\_\+\+DONT\+\_\+\+WANT\+\_\+\+TO\+\_\+\+CRASH\+\_\+\+ON\+\_\+\+ERROR\+\_\+\+I\+\_\+\+KNOW\+\_\+\+WHAT\+\_\+\+I\+\_\+\+AM\+\_\+\+DOING} to disable abort/exit on fatal errors.
\item Define {\ttfamily BLOP\+\_\+\+I\+\_\+\+DONT\+\_\+\+WANT\+\_\+\+TO\+\_\+\+CRASH\+\_\+\+ON\+\_\+\+ERROR\+\_\+\+I\+\_\+\+KNOW\+\_\+\+WHAT\+\_\+\+I\+\_\+\+AM\+\_\+\+DOING} to completely disable all BLOP assertions.
\item Define {\ttfamily BLOP\+\_\+\+SAFE\+\_\+\+MODE} to enable NULL pointer checks.
\item Define {\ttfamily BLOP\+\_\+\+ABORT\+\_\+\+ON\+\_\+\+ERROR} to use abort() instead of exit(-\/1) on assertion.
\end{DoxyItemize}

These flags allow each user to fine-\/tune BLOP\textquotesingle{}s safety vs performance tradeoffs.
