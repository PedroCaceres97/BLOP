Detailed Description

Generic intrusive doubly-linked list (header-only, macro-configurable).

This header implements a small, configurable, intrusive doubly-linked list. The module is designed to be lightweight, header-only, and fully customizable through macro configuration.

Overview

The list is intrusive: each node holds its own data and the pointers linking it to the list. This reduces allocations and allows users to fully control memory layouts.

The API is generated from configuration macros and may produce multiple list types simply by including this header with different macro definitions.

How to use

  • Include normally to get declarations.

  • In exactly one translation unit, define:

    #define BLOP_LIST_IMPLEMENTATION

    before including this header so that implementation bodies are emitted.

Example

#define BLOP_LIST_NAME mylist
#define BLOP_LIST_DATA_TYPE int
#define BLOP_LIST_IMPLEMENTATION
#include "list.h"
